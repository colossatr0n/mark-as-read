Regex Filters

Filters are useful when links add non-deterministic metadata or tracking data into the URL that changes every time the link is clicked.

Adding a filter to remove the non-deterministic metadata from a URL allows such URLs to be identified as a single URL, meaning the resulting page can be correctly marked as read.

These kinds of URLs tend to be generated when clicking marketplace/classified listings.
